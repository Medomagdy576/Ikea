# IKEA Website Automation Testing Framework

## 📌 Project Overview

This project is an end-to-end automation testing framework developed for the IKEA website using Selenium WebDriver, Cucumber, and Java.

The framework follows the Behavior Driven Development (BDD) approach to create readable, maintainable, and scalable test scenarios that simulate real user interactions across different functionalities of the website.

The project was built as part of my software testing portfolio to demonstrate practical experience in designing and implementing automated UI test frameworks.

## 🏗️ Framework Design

The project follows the **Page Object Model (POM)** design pattern to improve code reusability, readability, and maintainability.

### Project Structure

```text
src
├── test
│   ├── java
│   │   ├── Pages
│   │   ├── StepDefinitions
│   │   ├── runners
│   │   └── hooks
│   │
│   └── resources
│       └── features
│
└── Reports
```

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

### 2. Open the project

Open the project using IntelliJ IDEA or Eclipse.

### 3. Install dependencies

```bash
mvn clean install
```

### 4. Run the test suite

Execute the Cucumber Test Runner.
